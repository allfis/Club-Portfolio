# KUET Career Club Website

Welcome to the KUET Career Club (KCC) Website project.

This is a full-stack club management platform that I designed and developed to streamline the activities of KUET Career Club. The goal of this project was to create a centralized system where students can explore club events, register for activities, and stay connected with the club, while administrators can efficiently manage events and user interactions through a dedicated admin panel.

The website dynamically retrieves event information from a SQL Server database and presents it through a modern, responsive interface. Administrators can create, update, and manage events, upload event images, view registrations, and handle contact form submissions. Visitors can browse upcoming and past events, view detailed event information, and register for events seamlessly.

This project was developed using ASP.NET (C#), Entity Framework, SQL Server, HTML, CSS, and JavaScript.

## Key Features

### Public Features

* Browse upcoming and past club events
* View detailed event information
* Event registration system
* Contact form for visitors
* Responsive and user-friendly interface
* Dynamic event loading from the database
* Event image support
* Chatbot support

### Admin Features

* Secure admin authentication
* Create, edit, and delete events
* Upload and manage event images
* Control event visibility
* Manage event registrations
* View contact form submissions
* Administrative dashboard for content management

## Technologies Used

* ASP.NET (C#)
* Entity Framework
* SQL Server
* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API

## Purpose

This project was developed as a practical implementation of web development concepts, database management, and administrative content management systems. It demonstrates full-stack development skills, including frontend design, backend development, database integration, file handling, and dynamic content rendering.
